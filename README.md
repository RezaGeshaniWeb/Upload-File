# آپلود فایل با Node.js و Multiparty
  
یک پروژه ساده و آموزشی برای آپلود فایل با استفاده از `Node.js`، ماژول داخلی `http` و پکیج `multiparty` است.
در این پروژه، کاربر از طریق یک فرم HTML فایل خود را ارسال می کند و سرور فایل را به صورت استریم در پوشه `stream` ذخیره می کند.

## ویژگی ها

- راه اندازی سرور HTTP بدون فریم ورک (بدون Express)
- نمایش فرم آپلود فایل در مسیر اصلی (`/`)
- دریافت فایل به صورت `multipart/form-data`
- ذخیره فایل آپلودی به صورت استریم روی دیسک
- نمایش پیام موفقیت پس از پایان آپلود

## تکنولوژی های استفاده شده

- `Node.js`
- `http` (ماژول داخلی Node)
- `fs.createWriteStream` (برای ذخیره استریمی فایل)
- `path.extname` (برای استخراج پسوند فایل)
- `multiparty` (برای پردازش فرم چندبخشی)

## ساختار پروژه

```text
Upload-File/
├─ index.js
├─ package.json
└─ stream/        # فایل های آپلودشده در این مسیر ذخیره می شوند
```

> اگر پوشه `stream` وجود نداشته باشد، عملیات ذخیره فایل با خطا مواجه می شود.

## پیش نیازها

- نصب بودن `Node.js` (نسخه 14 یا بالاتر پیشنهاد می شود)
- نصب بودن `npm`

## نصب و اجرا

1. نصب وابستگی ها:

```bash
npm install
```

2. اجرای پروژه:

```bash
npm start
```

3. باز کردن برنامه در مرورگر:

```text
http://localhost:3000
```

## نحوه کار برنامه

1. با درخواست `GET` به مسیر `/`، یک فرم HTML ساده نمایش داده می شود.
2. کاربر یک فایل انتخاب و فرم را با متد `POST` ارسال می کند.
3. `multiparty` فایل را از بدنه درخواست استخراج می کند.
4. سرور با `createWriteStream` فایل را داخل پوشه `stream` ذخیره می کند.
5. بعد از اتمام ذخیره، پاسخ موفقیت شامل نام فایل به کاربر نمایش داده می شود.

## جزئیات فنی مهم

- پورت پیش فرض برنامه در `index.js` برابر `3000` است.
- نام فایل ذخیره شده با `Date.now()` تولید می شود تا احتمال تداخل نام کاهش یابد.
- پسوند فایل اصلی با `extname(part.filename)` استخراج و به نام نهایی اضافه می شود.

نمونه نام فایل ذخیره شده:

```text
1714032000000.jpg
```

## اسکریپت های npm
